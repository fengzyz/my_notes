## Nginx介绍
NGINX可用作HTTP / HTTPS服务器，反向代理服务器，邮件代理服务器，负载均衡器，TLS终结器或缓存服务器。它是相当模块化的设计。它具有由社区创建的本机模块和第三方模块。以C语言编写，它是一个非常快速和轻便的软件。
## Ubuntu 16.04 源码编译nginx
### 从源头构建NGINX的要求,强制性要求：
- OpenSSL库版本介于1.0.2 - 1.1.0之间
- Zlib库版本介于1.1.3 - 1.2.11之间
- PCRE库版本在4.4 - 8.40之间
- GCC编译器
### 可选要求：
- PERL
- LIBATOMIC_OPS
- LibFD
- MaxMind GeoIP
- libxml2的
- libxslt
